코딩을 좋아하는 웹 퍼블리셔 정우민입니다.
플래시와 액션스크립트로 코딩에 입문하였고
자연스럽게 웹 퍼블리싱에 관심을 가지게 되었습니다.
덕분에 웹에서도 인터랙션과 모션을 구현하는 것을
좋아하게 되었고, 올바르게 작성된 코드 위에
동적인 인터랙션을 추가하는 것이 특기입니다.
인터랙션에 치중하여 기초적인 부분을 놓치는 것이 아닌
웹 표준과 웹 접근성을 지켜서 마크업합니다.
저는 코딩을 좋아하고 사랑하기 때문에
퍼블리셔로 머물지 않고 프론트엔드 개발자가 되기 위해
더욱 공부하고 정진합니다.
